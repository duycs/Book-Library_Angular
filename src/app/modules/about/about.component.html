Librarians can does:

  • Register
  • Logout
  • View all books
  • Add new book
  • In the list books you choose a book then edit or delete this item
  • View all book items
  • Add new book item
  • In the list book items you choose a book then edit or delete this item
  • View all members
  • In the list members items you choose a member then block or unblock this item

Members can does:

  • Register
  • Logout
  • Checkout book
  • Reserve book
  • Renew a book
  • Return a book
